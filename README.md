# Amazon E-commerce Sales Analysis

## Project Overview
This project analyzes an Amazon e-commerce sales dataset containing around 129k orders. The objective is to understand sales trends, product performance, customer behavior, and shipping impact using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
Amazon sales dataset with order details such as order ID, category, shipping information, order status, and revenue.

## Key Insights
- Maharashtra generates the highest revenue.
- Sales peaked in April.
- Cancellation rate is approximately 8.8%.
- The Set category dominates product sales.
- Expedited shipping contributes around 73% of revenue.
- The business is primarily B2C.

## Conclusion
The analysis helps identify key sales trends and operational insights that can help businesses optimize inventory, shipping strategies, and customer experience.
